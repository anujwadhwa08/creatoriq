CreatorIQ

YouTube Creator Intelligence Platform  ·  Agentic AI  ·  Phase 1

What We Built
The Problem
YouTube creators publish without knowing what works. There is no plain-English intelligence system that looks at your channel's real data and tells you what to do next — when to post, which topics win, how original your content is, and what your next video will likely earn. Existing tools surface raw numbers. None of them synthesise, prioritise, or explain.

Core Capabilities
Capability What it does
Automated Reports Three cadences: velocity check (every 4 h after upload), daily digest, full weekly brief — all determined by the scheduler, never hardcoded
Performance Agent Analyses views/day, engagement %, CTR, watch time, and duration buckets — uses views_per_day not raw view_count to eliminate age bias
Audience Agent Sentiment analysis on real YouTube comments; surfaces praise themes, frustration signals, and unanswered audience questions
SEO & Timing Agent Extracts top recurring keywords, tag coverage gaps, empirically best posting day and hours from historical data — weekly reports only
Prediction Agent RAG retrieval of 5 similar past videos from ChromaDB + Pearson correlations → predicted views, likes, and estimated revenue for any planned video
Originality Scorer Sentence embeddings (all-MiniLM-L6-v2) + ChromaDB vector search across the full 59-creator corpus — surfaces the most similar video from any other creator
Synthesis Agent Combines all parallel agent outputs into a single narrative brief with ranked action items via Claude Sonnet 3.5, typed JSON schema output
LLM Judge A separate Claude call independently scores every brief on 5 quality dimensions per report type — Specificity, Actionability, Accuracy, Clarity, Trend Honesty
Claude Desktop MCP FastMCP server exposes 8 tools + 2 resources — any creator's data can be queried directly from Claude Desktop without opening the dashboard
